## 2. 📐 Visual System
*Adhere strictly to `GENERAL/G1 - Styles.md`.*

- **Core Accent:** Warm Amber (`#D4923A` in dark mode, `#C47A2D` in light mode) implemented via standard system `CurrentColor` framework operations.
- **Corner Radii:** 14px outer shells, 10px inner shells.
